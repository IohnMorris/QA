# ✅ Позитивные тест-кейсы: Форма Text Box

**Объект тестирования:** Элемент UI `Text Box`

**Тестовая среда:** [DemoQA Text Box](https://demoqa.com/text-box)

**Предусловие для всех тестов:** Открыта страница браузера по указанному URL.

**Цель:** Проверка корректной работы системы при вводе валидных данных, обработки граничных значений и необязательных полей.

| ID | Название (Title) | Шаги (Steps) | Ожидаемый результат (Expected Result) |
| :--- | :--- | :--- | :--- |
| **TC-TB-001** | **Стандартное заполнение всех полей (Happy Path)** | 1. Ввести `Ivan Ivanov` в **Full Name**.<br>2. Ввести `ivan@mail.com` в **Email**.<br>3. Ввести `Moscow, Tverskaya 1` в **Current Address**.<br>4. Ввести `Moscow, Arbat 10` в **Permanent Address**.<br>5. Нажать **Submit**. | Форма успешно отправляется. Под кнопкой появляется блок с результатами, значения полностью соответствуют введенным данным. |
| **TC-TB-002** | **Граничные значения (Минимально допустимый ввод)** | 1. Ввести `A` в **Full Name**.<br>2. Ввести `a@b.co` в **Email**.<br>3. Ввести `B` в **Current Address**.<br>4. Ввести `C` в **Permanent Address**.<br>5. Нажать **Submit**. | Форма успешно отправляется. В блоке результатов корректно отображаются введенные одиночные символы и короткий email. |
| **TC-TB-003** | **Поддержка UTF-8, кириллицы и спецсимволов** | 1. Ввести `Иван Иванов Ё` в **Full Name**.<br>2. Ввести `test@test.com` в **Email**.<br>3. Ввести `Улица №5, кв. 10! @#$%` в **Current Address**.<br>4. Ввести `🏠 Дом 🌍` в **Permanent Address**.<br>5. Нажать **Submit**. | Форма успешно отправляется. Система корректно обрабатывает кириллицу, спецсимволы и эмодзи. Данные не искажаются (отсутствуют "кракозябры"). |
| **TC-TB-004** | **Отправка формы только с обязательными полями** | 1. Оставить **Full Name** пустым.<br>2. Оставить **Email** пустым.<br>3. Ввести `Moscow, Tverskaya 1` в **Current Address**.<br>4. Оставить **Permanent Address** пустым.<br>5. Нажать **Submit**. | Форма успешно отправляется. Система обрабатывает пустые значения без вызова ошибок. В блоке результатов отображается только заполненное поле `Current Address`. |
| **TC-TB-005** | **Граничные значения (Максимально допустимый ввод)** | 1. Ввести строку из ровно **255** символов в **Full Name**.<br>2. Ввести `ivan@mail.com` в **Email**.<br>3. Ввести строку из ровно **1000** символов в **Current Address**.<br>4. Ввести строку из ровно **1000** символов в **Permanent Address**.<br>5. Нажать **Submit**. | Форма успешно отправляется. Под формой отображаются все строки в полном объеме (без обрезания символов). БД успешно принимает предельно допустимые значения. |
